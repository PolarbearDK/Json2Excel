# Json2Excel

Convert json file with arrays to Excel

```
Convert Json file to excel file.

Usage: Json2Excel -Input:<Input> [-Output:<OutputFileName>]
[-Extension:<OutputExtension>] [-Sample:<Sample>] [-Worksheet:<Worksheet>]
[-Help]

-Input:<Input> (Alias: -I)
    Input json file.

-Output:<OutputFileName> (Alias: -O)
    Output excel file. Default: Import file with extension specified in
    Extension argument

-Extension:<OutputExtension> (Alias: -E)
    Output excel file. Default: Import file with .xlsx extension

-Sample:<Sample> (Alias: -S)
    Number of rows to sample for json properties. Default: all rows

-Worksheet:<Worksheet> (Alias: -W)
    Name of worksheet. Default is the filename of the input file.

-Help (Alias: -H/-?)
    Show help.
```
